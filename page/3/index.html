72.Edit Distance

发表于 2016-12-15 | 分类于 LeetCode

题目链接！

解题思路

  • 使用DP动态规划，dp[i][j]表示word1中的前i个字符与word2中的前j个字符的距离；
  • dp[i][j]的可能取值如下：
  • word1插入一个字符：dp[i][j-1] + 1;
  • word1删掉一个字符：dp[i-1][j] + 1;
  • 两字符串最后一位不相同，则替换word1的最后一个：dp[i-1][j-1] + 1;
  • 如果相同，则不需要操作：dp[i][j];
  • DP取上述情形的最小值。
