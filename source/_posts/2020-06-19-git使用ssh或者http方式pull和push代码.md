切换成http方式：

```bash
git remote set-url origin https://github.com/haoyuanliu/haoyuanliu.github.com.git
```

切换成ssh方式：

```bash
git remote set-url origin git@github.com:haoyuanliu/haoyuanliu.github.com.git
```
