74. Search a 2D Matrix

解题思路

  • 先找到target所在的行，然后对所在行进行二分查找，结果超时；
  • 将矩阵看做一个一维数组，然后整体使用二分查找；

240. Search a 2D Matrix II

解题思路

  • 从右上角开始查找；
  • 如果小于target，则这一行全部小于target；
  • 如果大于target，则这一列全部大于target；
  • 按照这个策略进行搜索即可；
